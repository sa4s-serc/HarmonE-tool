# API Reference

This directory contains technical documentation for the interfaces exposed by Harmonica.

## Contents

1.  **[REST API Reference](rest_endpoints.md)**
    *   Describes the HTTP endpoints exposed by the **Managing System (ACP)** and the **Managed System Wrapper**.
    *   Use this if you are building a new Frontend or integrating an external monitoring tool.

2.  **[Python Interface Reference](python_interfaces.md)**
    *   Describes the internal Python functions required to implement a **Custom Managed System**.
    *   Use this if you are a researcher adding a new ML pipeline (e.g., NLP, Audio Processing) and want to define your own MAPE logic.
